How It Works

Backend: Flask app authenticates with Google Analytics using a service account, queries the GA4 Data API for active user metrics, and formats the response as JSON.

Frontend: Chart.js renders the data with custom styling. The 30-day window automatically shifts to exclude the most recent 48 hours (GA4's standard processing time).

Deployment: Hosted on Render with environment variables storing API credentials securely.
